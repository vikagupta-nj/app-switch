Available Demos

This demo suite shows different approaches to detecting WebView environments:

Client-Side Detection

JavaScript
Browser-Based

Detects WebViews using JavaScript directly in the browser by analyzing:

  • User agent strings
  • Bridge objects
  • Missing browser features
  • App-specific patterns
No server component required. Works entirely in the browser.

Server-Side Detection

Node.js
HTTP-Based

Detects WebViews on the server by analyzing:

  • HTTP headers
  • User agent patterns
  • Request characteristics
  • Missing browser headers
Server component required. Can adapt content before sending to client.
